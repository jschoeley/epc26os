# Lecture on "Open Science Practices for Demographic Research" at European Population Conference (EPC) 2026 at EAPS Working Group on Open Science in Demography

Materials, slides, hands-on code for the lecture on "Open Science Practices for Demographic Research" covering topics such as Open Science, Git, GitHub, Version control, etc.

Here you can find more information about the [EPC 2026 conference program](https://epc2026.eaps.nl/summary) and the [EAPS Working Group on Open Science in Demography](https://www.eaps.nl/page/open-science-in-demography).

Here is the folder structure and what to expect:

- Root folder
    - This ReadMe.md text file
    - File named ".gitignore" is very important and described how it works in the lecture slides.
- Slides
    - Download PDF file of the lecture slides.
- Hands_on
    - We have an interactive, self-paced course on `Git, GitHub, Version control`. Follow the instructions in this folder to start.
    - You need to have a `GitHub` account to follow this interactive course. **Sign up here**: [https://github.com/signup?](https://github.com/signup?)
